Boolean Logic

AND: Each result contains all the search terms

OR: Each result contains at least one search term

NOT: None of the results contain the specified term

Wildcards and Truncation

Each databases handles wildcards in unique ways

* Asterisk: most often used for truncation
? Question mark: used for unknown character
# Hashtag: use for alternate spellings

  • comput*


    computer, computers, computing, computation, computations, computational

  • d?g


    dig, dog, dug

    an * can often be used as a "hungry" wildcard in the middle of a word d*g = drug

  • counsel#or


    counselor, counsellor
